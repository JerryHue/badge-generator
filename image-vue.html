Note that width and height are optional, but they can help in certain cases. For example, a screenshot or logo will be set by GitHub to 100% of the page width making it look large and take up vertical space, so reduce height or width manually. Setting heigh can also make sure the height of the tag is set before the image loads, so it reserves the space - if the height is larger than the original image though then the image will looked stretched.
